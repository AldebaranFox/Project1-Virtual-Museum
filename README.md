
To access the website do the following:
   1. Have "Live Server" extension installed
   2. Open terminal
   3. Type "git clone git@github.com:AldebaranFox/Project1-Virtual-Museum.git"
   4. Go to the directory
   5. Open "index.html" (inside "HTML" folder)
   6. Bottom right corner of screen slick "Go Live"
   7. Explore the website
